## General info
E-Shop is an e-commerce web app tyhat allows users to view, add products to their cart and purchase products using paypal. The application is built using the MERN stack.
	
## Technologies

Front-End:
* React JS
* React-Bootstrap
* React-Redux
* Redux-thunk
* Axios

Back-End:
* Node JS
* Express JS
* Bcrypt
* Json Web Token
* Mongo DB 
* Mongoose
* Multer
